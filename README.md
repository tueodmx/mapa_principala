# mapa_principala
**Descriere proiect GitHub:**

Proiectul presupune utilizarea GitHub pentru a crea și gestiona depozite, a încărca fișiere, a lucra cu ramuri și Pull Requests, și a colabora eficient în echipă. Se va învăța cum să se creeze un depozit, să se adauge fișiere de laborator (PDF, DOCX, PNG), să se includă linkuri către prototipuri Figma și să se formateze un fișier README.md pentru documentarea proiectului. De asemenea, se vor utiliza Issues pentru urmărirea sarcinilor și feedback-ului, iar colaborarea va fi facilitată prin comentarii și revizuiri ale modificărilor.

Scopul este să se învețe gestionarea unui proiect pe GitHub, organizarea fișierelor, colaborarea cu alți membri și managementul modificărilor într-un mod structurat.
